# DefFunc

Unity 6 Tank Prototype Game
This is a simple Unity 6 tank prototype. 
The game includes features like movement, shooting,health management, soundFX and more, for both the player and enemies.
It includes 2 Types of Enemies: Melee Enemy & Range Enemy.
It uses Unity’s Behavior Graph for Controlling the Enemies.

If you want to improve it or fix stuff, feel free to do so!
